1. 這款遊戲是給 9 到 12 歲的小朋友使用的
2. 主要用途為練習系動詞與助動詞
3. 程式是由 Claude AI 開發
4. 共 4 個關卡設計，關卡如下：
  - 關卡 1 — 填空題：句子中有空格，讓小朋友從選項中選出正確的 be 動詞或助動詞（am / is / are / was / were / can / will / do / does...）
  - 關卡 2 — 句子重組：把打亂的單字卡拖拉排列成正確句子
  - 關卡 3 — 對話練習：給一段對話情境，小朋友選擇適當的回應（類似選擇題，但包在對話框裡）
  - 關卡 4 — 闖關遊戲：綜合前三種，加上倒數計時、生命值、最終計分板
